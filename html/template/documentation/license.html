Licensing in RS/JSON is a way to inform the end developer how to treat the schema. Depending the license, he will be able to use it for commercial use, inherit and update the schema, and others use cases. The license attribute is just a license name. No URL is required. You may also put your company name if it is a proprietary license.
{ "version": "0.1.0", "license": "MIT", "charset": "utf8", "schema": {} }
