version

  • type string
A version number. To work better, mostly with migrations, you should support semantic versioning.

license

  • type string
A license name.

charset

  • type string
A character set to use for all the property inside the schema. See the charsets documentation for a list of supported character set.

schema

  • type object
Each key represent a model name

model

  • type object
A model represent a table in a database.

identifier

  • type array<string>
A list of property name to use to identify the record.

uniques

  • type array<unique>
A list of unique objects to use to add a constraint of the desired properties.

unique

  • type array<string>
A list of property name to use to specify the unique index.

properties

  • type object
A list of property.
