Custom types extend from native types to abstract some logics. They take a very good place when it comes to scale and improve maintenance.

However, this is at you, developer, to either import them from the community or create them in order to use some.

Here is an example that shows how to create a custom type: longitude.

  • PHP
  • Node 

use RsJson\RsJson;
use RsJson\Property;

if (!RsJson::hasType('longitude')) {
  RsJson::addType('longitude', function(Property $property): void {
    $property->type = 'float';
    $property->precision = 8;
    $property->guards = ['min:-180', 'max:180'];
  });
}
        

const RsJson = require('rs-json');
// import * as RsJson from 'rs-json';

if(!RsJson.hasType('longitude')) {
  RsJson.addType('longitude', property => {
    property.type = 'float';
    property.precision = 8;
    property.guards = ['min:-180', 'max:180'];
  });
}
