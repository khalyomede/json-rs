RS/JSON supports a list of charsets in order to unify every variants on the plaform RS/JSON will be installed.
Name
utf8
utf8mb4
iso88591
Charsets can be set globally:
{ "version": "0.1.0", "license" : "MIT", "charset": "utf8", "schema": {} }
Or per property:
{ "version": "0.1.0", "license": "MIT", "charset": "utf8", "schema": { "book": { "properties": { "name": { "type": "string", "charset": "utf8mb4" } } } } }
In such case, a charset set on any property will override the default charset.
